# Travel-Guide Website 🧳✈️🗽
This is a Travel Guide Website, built using html, css, JS, and PHP.

## Features
- Registration.
- View available locatation to travel to.
- Having different packages available for each location.
- Book packages.
- View booked packages.

## Technologies
- html
- css
- Javascript
- php


## How to run the website
1. Clone to the repository into a folder.
2. Open the folder in a code editor (VS Code preferred).
4. Download 'Live Server' extension.
5. Right click on 'index.html', then click on 'Run with Live Server'.
